# 新建歌单功能说明

## 功能概述
本功能允许用户创建新的歌单，包括歌单名称、简介和封面图片。

## 功能特性

### 1. 歌单信息输入
- **歌单名称**: 必填项，不能为空
- **歌单简介**: 必填项，不能为空
- **封面图片**: 可选，支持JPG和PNG格式

### 2. 图片上传功能
- **支持格式**: JPG、JPEG、PNG
- **文件大小限制**: 最大2MB
- **上传路径**: D:/下载/image
- **默认封面**: 如果用户未上传图片，将使用默认封面图片

### 3. 表单验证
- 前端实时验证用户输入
- 后端验证文件格式和大小
- 完整的错误提示信息

## 使用方法

### 1. 访问我的页面
在应用主界面点击"我的"标签页

### 2. 创建歌单
1. 点击"新建歌单"标签
2. 在弹出的对话框中填写歌单信息：
   - 输入歌单名称
   - 输入歌单简介
   - 可选择上传封面图片（拖拽或点击上传）
3. 点击"创建歌单"按钮

### 3. 查看创建的歌单
创建成功后，歌单将出现在"我的歌单"列表中

## 技术实现

### 前端 (Vue 3 + Element Plus)
- 使用Element Plus的Dialog、Form、Upload组件
- 支持拖拽上传图片
- 实时预览上传的图片
- 表单验证和错误处理

### 后端 (Spring Boot)
- 文件上传API: `/users/upload`
- 创建歌单API: `/music/createPlaylist`
- 文件格式和大小验证
- 自动生成唯一文件名

### 数据库
- 歌单信息存储在playlist表中
- 字段包括：id、name、introduction、createId、image

## 注意事项

1. **文件格式**: 只支持JPG和PNG格式的图片
2. **文件大小**: 图片文件不能超过2MB
3. **网络连接**: 上传图片需要稳定的网络连接
4. **存储空间**: 确保D:/下载/image目录有足够的存储空间

## 错误处理

- **文件格式错误**: 提示"只支持JPG和PNG格式的图片"
- **文件过大**: 提示"文件大小不能超过2MB"
- **网络错误**: 提示"图片上传失败，请重试"
- **表单验证**: 提示"请填写完整信息"

## 开发环境

- 前端: Vue 3 + TypeScript + Element Plus
- 后端: Spring Boot + MyBatis Plus
- 数据库: MySQL
- 文件存储: 本地文件系统 (D:/下载/image)
